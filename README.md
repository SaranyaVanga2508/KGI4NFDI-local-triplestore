# KGI4NFDI Local Triple Store

This project demonstrates a local RDF triple store with a validation layer for importing the KGI4NFDI Knowledge Graph Registry.

## Features

- Local Apache Jena Fuseki triple store
- RDF validation
- Dataset import
- SPARQL querying
- Import completeness verification

## Tech Stack

- Python
- Apache Jena Fuseki
- RDFLib
- Docker
- SPARQL
